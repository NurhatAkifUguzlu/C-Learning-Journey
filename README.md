# Fibonacci Calculator (Recursive)

This is a simple C program that calculates the Fibonacci number at a specific index provided by the user.

## How it Works
It uses a **recursive function** logic.
- Base Case: Returns 0 or 1 for the first two indices.
- Recursive Case: Returns `fib(n-1) + fib(n-2)`.

## Usage
1. Compile the code using GCC: `gcc main.c -o fibonacci`
2. Run the program: `./fibonacci`
3. Enter an index number.
